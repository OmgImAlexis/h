# h
Hosted apps done simply.

## Installation

```bash
git clone https://github.com/omgimalexis/h ~/code/h && cd ~/code/h

# Setup server
./admin/install user@server-domain.tld

# Setup app on server
./admin/create user@server-domain.tld demo

# Deploy app
cd ~/code/demo
git push production master

# Profit ?
```
